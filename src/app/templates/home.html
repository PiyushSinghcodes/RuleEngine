Welcome to the Rule Engine API

This API allows you to create, combine, and evaluate rules.

Available Endpoints:

  • Create Rule: /create_rule

    Create a new rule by providing a rule string.

  • Combine Rules: /combine_rules

    Combine multiple rules into a single rule.

  • Evaluate Rule: /evaluate_rule

    Evaluate a rule against provided data to get the result.

  • Health Check: /health

    Check the health status of the API.
